# CodeAlpha - Basic Chatbot

## Project Description
This project was developed as part of the **CodeAlpha Python Programming Internship**.

It is a simple rule-based chatbot built using Python. The chatbot responds to predefined user inputs such as greetings, basic questions, jokes, date, time, and calculator operations.

## Features
- Greeting responses
- Date and Time
- Simple Calculator
- Random Jokes
- Motivational Quotes
- Chat History
- Rule-based conversation
- Modular Python code

## Technologies Used
- Python 3

1. Download or clone the repository.
